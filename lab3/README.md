ARTI308 Lab 3
Overview
This lab focuses on analyzing a cars dataset and applying machine learning techniques to predict car prices. The goal is to clean the data, explore it, and build a predictive model.
Dataset
The dataset used in this lab is global_cars_enhanced.xlsx, which contains information about different cars such as brand, engine size, horsepower, fuel type, and price.
Tasks
Task 1 – Data Preprocessing
The dataset was loaded using Pandas, and preprocessing steps were applied such as handling missing values, removing duplicates, and converting data into the correct format.
Task 2 – Data Exploration
Exploratory Data Analysis (EDA) was performed to understand the relationships between variables. Visualizations were used to identify patterns between features like engine size, horsepower, and price.
Task 3 – Feature Selection
Relevant features were selected to improve the model’s performance. Irrelevant or less important features were removed to reduce noise in the data.
Task 4 – Model Building
A machine learning model (Linear Regression) was built to predict car prices based on selected features. The dataset was split into training and testing sets.
Task 5 – Model Evaluation
The model was evaluated using metrics such as Mean Squared Error (MSE) and R² score to measure its accuracy and performance.
